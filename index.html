Learn javascript with basic projects

When I began learning JavaScript fundamentals, I struggled to find useful tutorials that included basic JavaScript projects. Understanding the basics and observing them in action is crucial for thinking like a programmer and comprehending the chain of changes required for successful interventions. My suggestion is to start by reading JavaScript fundamentals thoroughly. Once you have a good grasp of the basics, you can try the projects I've provided. If you find those projects too challenging, you can also find simpler projects on my GitHub page to start with
